# Portfolio

Files for my personal portfolio, built with [React](https://www.reactjs.org/) and [Tailwind CSS](https://tailwindcss.com/).

## Scripts

To run:

```sh
$ yarn start
```

To test with [Jest](https://jestjs.io/):

```sh
$ yarn test
```

To build:

```sh
$ yarn build
```
